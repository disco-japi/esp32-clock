Proyecto con propositos demostrativos y educativos

Reloj analogico y digital creado haciendo uso de:
- ESP32
- Módulo RTC DS3231
- Módulo pantalla OLED SH1106

Pinout:

| Periferico | PIN          |
| :------------------- | :----------: | 
| SDA (RTC y OLED, en paralelo)            | D21     | 
| SCK (RTC y OLED, en paralelo)             | D22      | 
| Interruptor modo tono negativo           | D34      | 
| Interruptor modo analógico      | D23      |
